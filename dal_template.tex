\begin{dal}{} %Az üres pár kapcsos zárójel közé írd be a címez
	\vsz{
		% Ide írd az első versszakot úgy, hogy az utolsó sor kivételével minden sor végén legyen '\\'. Például:
		%
		% Erdő, erdő kerek erdő\\
		% De szép madár lakja kettő.\\
		% Kék a lába, szöld a szárnya\\
		% Piros a rózsám orcája
	}

	% Ha egynél több versszakos a dal, akkor minden verszakot egy olyan `\vsz{`-szel kezdődő blokkba írj, mint az előzőt. Ne felejtsd el a blokkot kapcsos zárójellel lezárni (`}`).
\end{dal}
